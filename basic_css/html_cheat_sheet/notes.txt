Inline vs Block Level Elements
Inline elements
  Do no start a new line
  Take only the necessary width

Block Elements
  Start on a new line
  Take full width available

  Block Level: <div>, <h1>--<h6>, <form>
  Inline Level: <span>, <img>, <a>
